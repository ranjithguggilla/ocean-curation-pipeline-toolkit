# Sample Data Documentation

## Included Files

### 1. `raw/GOMECC4_cruise_watersamples_RAW.xlsx`
**Type:** Excel workbook (intentionally messy)
**Description:** Simulated water chemistry measurements from a GOMECC-4-style
Gulf of Mexico research cruise. Contains 63 observation rows at 8 stations
across 8 depth levels, plus intentional data quality issues.

**Intentional Issues (for pipeline demonstration):**
- Mixed date/time formats across rows (MM/DD/YYYY, YYYY-MM-DD, DD-Mon-YYYY, etc.)
- Whitespace artifacts in station identifiers
- ~5% missing values in dissolved oxygen, pH, and nutrient columns
- 3 exact duplicate observation rows
- 1 completely empty row
- Non-standard column headers with special characters (°, µ)
- Hidden "Metadata" sheet

### 2. `raw/station_log.csv`
**Type:** CSV (clean but needs header normalization)
**Description:** Station occupation log with coordinates, bottom depth, CTD
serial number, weather conditions, and field notes for each of the 8
sampling stations.

### 3. `raw/station_data.csv`
**Type:** CSV (backward-compatible clean version)
**Description:** Simplified version with 32 rows (8 stations × 4 depths).
Used by the test suite.

## Data Generation

All sample data is synthetic, generated by `generate_samples.py`. The
oceanographic values (temperature profiles, salinity, dissolved oxygen) use
realistic Gulf of Mexico parametric models to produce plausible but not
real observations.

**To regenerate:** `python sample_data/generate_samples.py`

## Real Data Sources for Production Use

For actual submissions, replace these files with real datasets from:

1. **NOAA NCEI GOMECC-4:** https://www.ncei.noaa.gov/access/ocean-carbon-acidification-data-system/
2. **NOAA NCEI Archive:** https://www.ncei.noaa.gov/
3. **NOAA NDBC Gulf Buoys:** https://www.ndbc.noaa.gov/
4. **IOOS Glider DAC:** https://gliders.ioos.us/
